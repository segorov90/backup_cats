# Резервное копирование картинок с котиками

Программа для резервного копирования картинок с котиками и пользовательским текстом на Яндекс.Диск.

## Функциональность

- Получение картинок котиков с пользовательским текстом с сайта cataas.com
- Создание папки на Яндекс.Диске с названием группы в Нетологии
- Загрузка картинок напрямую на Яндекс.Диска (без локального сохранения)
- Сохранение информации (локально) о загруженных файлах в JSON-файл
- Логирование всех операций в файл backup.log
- Прогресс-бар для отслеживания процесса загрузки
